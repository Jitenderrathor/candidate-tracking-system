# Candidate Tracking System Documentation

This directory is the source of truth for system-level documentation.

- [`architecture/README.md`](architecture/README.md) describes backend boundaries, request flow, and extension rules.
- [`api/README.md`](api/README.md) records the current HTTP surface and response contracts.
- [`decisions/README.md`](decisions/README.md) explains how architecture decisions should be recorded.
- [`INSTALLATION.md`](INSTALLATION.md) covers local and release installation.
- [`ENVIRONMENT.md`](ENVIRONMENT.md) documents runtime and build-time configuration.
- [`DEPLOYMENT.md`](DEPLOYMENT.md) defines production topology and rollout steps.
- [`PRODUCTION_CHECKLIST.md`](PRODUCTION_CHECKLIST.md) is the release gate.
